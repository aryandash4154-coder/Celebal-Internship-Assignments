# SQL Sales Analysis Assignment

## About This Project

This project focuses on analyzing real-world sales data using SQL. The dataset is based on the Superstore Sales dataset from Kaggle. All SQL queries were written and tested using MySQL Workbench 8.0.

**Name:** Aryan Dash

**Tool:** MySQL Workbench 8.0
**Dataset:** Custom dataset created based on the Kaggle Superstore dataset structure. The schema was self-designed with relevant columns for SQL analysis.

## Project Overview

In this assignment, I analyzed a sales dataset containing thousands of customer orders. Using SQL queries, I extracted meaningful business insights such as identifying the highest-performing regions, the most profitable product categories, top customers, monthly sales trends, and the impact of discounts on sales performance.

## Key Findings

* The West region generated the highest total sales.
* The Technology category was the most profitable.
* The Consumer segment contributed the highest revenue.
* Higher discounts did not always result in higher sales.
